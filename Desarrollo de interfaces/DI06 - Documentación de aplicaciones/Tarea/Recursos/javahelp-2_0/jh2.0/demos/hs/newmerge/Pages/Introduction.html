Merge Demo

This merge demo is included with the release to demonstrate the JavaHelp system helpset merging capabilities.  

Merging Rules

The merging rules depend on the view's merge type. There are four merge types in JavaHelp:

Merge Type Description
sort Collate with existing view data according to helpset's locale collation rules.
append Add to the end of the existing view data.
unite-append  Unite any elements, including sub-elements, at the same level in the merged helpset with elements, including sub-elements, that have the same name in the initial helpset. Append remaining elements in the merged helpset at the end of the initial helpset. Add to the end of existing view data.
none No merging is done. Only valid on a view.
