db4o Replication System

The db4o Replication System provides functionality to periodically synchronize databases that work disconnected from each other. For example mobile devices synchronizing to a central server or desktop application.

The dRS can replicate between any number of db4o-databases. See "db4o to db4o Replication"

Another feature is to replicate between VODVersant Object Database and db4o. See "Replication With VOD"

Utilizing Hibernate dRS can also replicate between db4o and a RDBMSRelational Database Management System. See "Replication With Hibernate"

Before you start, please make sure that you have downloaded the latest dRS distribution from the download area of the db4o developer website.
