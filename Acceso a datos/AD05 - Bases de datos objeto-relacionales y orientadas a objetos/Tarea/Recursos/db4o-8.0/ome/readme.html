The zip file in this folder contains the Eclipse plugin version of Object Manager Enterprise, an object browser for db4o databases.

To install the plugin, you need to have a version of Eclipse >= 3.3 installed. Unzip the file to a folder of your choice. Then open Eclipse, select 'Help' -> 'Software Updates...' -> 'Available Software' from the menu. Choose 'Add Site...' -> 'Local...' and select the unzip folder. Follow the Eclipse Update Manager instructions for the OME feature from here on.

The actual menu structure may vary over Eclipse versions. (The above applies to Eclipse 3.4 Ganymede.) When in doubt, please refer to the Eclipse documentation on Software Updates.

Alternatively, you can install the plugin manually by simply copying the contents of the 'plugins' and 'features' folders from the unzip folder to the corresponding subfolders in the root folder of your Eclipse installation.
